# Post-deploy SEO checklist

After every deploy:
1. Go to Google Search Console → Sitemaps → resubmit https://studenzbit.com/sitemap.xml
2. Use URL Inspection tool to request indexing for each new page
3. Check Coverage report after 48hrs for any errors
